Wireless Biometric Lock Using Arduino:


  • Domain: Internet of things.



  • Role: Team Leader, Hardware Builder.



  • Description:


In this project we used Bluetooth module and ARDUINO as input and output devices in my project. When we place a finger on the BIOMETRIC SCREEN it detects whether finger matches to original or not and ensures the lock is opened for correct finger prints. Here we used SERVOMOTOR instead of door. When the finger is detected correctly the SERVOMOTOR rotates 180 degrees, if not it retains its original position. We used ARDUINO software to run and dump the code.
